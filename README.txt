EXECUTION DU JEU

Le jeu est fourni précompilé, et tous les fichiers binaires se trouvent dans le répertoire "bin/".
Un fichier nommé "exec.sh" a également été mis à disposition pour simplifier son utilisation. 
Pour démarrer le jeu, lancez l'executable "exec.sh"

Le jeu est compatible avec la version 21 de Java ou supérieure.
